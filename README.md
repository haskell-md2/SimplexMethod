# Лабораторная 1. Симплекс-метод

*Что нужно сделать, помимо отчёта:*

Придумать задачу, включающую 
5 переменных 3 неравенства 1 равенства
3 имеют ограничение на знак (неотрицательные)
Отсутствуют нулевые коэффициенты 

1. Форма постановки задачи в симметрической форме: 
Может взять 3 переменных и 3 неравенства

2. Привести все остальные

3. Ко всем вывести двойственную задачу

4. Исходную задачу приводим к виду, необходимую для симплекс метода (каноническая форма)

5. Решить симплекс методом с выбором начального приближения методом искусственного базиса прямую и двойственную задачи 

6. Если надо вывести параметр, то сразу могли это сделать (удобный вывод параметров и промежуточных результатов, например, невязка)

7. Предусмотреть вывод информации о смене базиса для вырожденного опорного вектора

8. (*)Процедура отстраивания от зацикливания и показать на отдельном примере

9. Решить ту же задачу методом перебора крайних точек с пояснением выбора алгоритма решение СЛАУ
